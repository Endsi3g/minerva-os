## 🏷️ Tags de Status & Typologie

| Tag | Usage |
| :--- | :--- |
| `#status/draft` | Document de travail en cours de rédaction. |

| `#status/approved` | Validé, fait foi pour l'implémentation. |

| `#type/spec` | Spécification technique détaillée d'une fonctionnalité. |
| `#type/process` | Tutoriel, workflow interne ou standard de dev. |
